# Programming Fundamentals Assignment 3

This repository contains my C++ solutions for Assignment 3 of the Programming Fundamentals course. The programs demonstrate the implementation of core programming concepts through a collection of problem-solving exercises.

## Repository Structure

```
Programming-Fundamentals-Assignments/
├── Assignment3-part1.cpp
├── Assignment3-part2.cpp
├── Assignment3-part3.cpp
└── Assignment3-part4.cpp
```

## Concepts Covered

- Functions
- Arrays
- Two-Dimensional Arrays
- Character Arrays
- Function Overloading
- Linear Search
- Bubble Sort
- String Manipulation
- Conditional Statements
- Loops

## Development Environment

- Language: C++
- IDE: Visual Studio Code
- Compiler: GCC / MinGW
